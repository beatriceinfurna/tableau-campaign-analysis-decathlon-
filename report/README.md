# Project Report

This folder contains the final deliverables of the Tableau campaign analysis project developed as part of the **IESEG Data Visualization course**.

---

## Contents

### Decathlon X Sissy presentation.pdf
The final **group presentation** delivered for the course.

The presentation includes:
- Business context and campaign objectives
- Dashboard structure and design rationale
- Key insights derived from the campaign analysis
- Strategic and business-oriented recommendations

### visualizations/
This subfolder contains selected **static visual outputs** extracted from the Tableau dashboards.

These figures:
- Support the insights discussed in the presentation
- Illustrate key trends, KPIs, and customer behavior
- Are included for documentation and portfolio purposes

### README.md
This file provides an overview of the contents of the `report` folder.

---

## Purpose

The material in this folder summarizes the **analytical outcomes and visual storytelling** of the project.

It is intended to:
- Complement the repository-level README
- Provide reviewers with direct access to the final presentation and visual results
- Showcase the dashboard logic and insights without requiring Tableau Public access
